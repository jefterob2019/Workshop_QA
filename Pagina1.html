Página feita para o projeto do Workshop de QA

Essa página é voltada para orientação sobre a carreira de QA em uma empresa



Nessa sessão descrevemos as principais ferramentas



  • Postman

    • O Postman é uma ferramenta que dá suporte à documentação das requisições feitas pela API.
    Ele possui ambiente para a documentação, execução de testes de APIs e requisições em geral.



  • Swagger

    • Trata-se de uma aplicação open source que auxilia desenvolvedores nos
    processos de definir, criar, documentar e consumir APIs REST



  • Rest Assured

    • O Rest Assured nos permite testar serviços RESTful em Java de um jeito muito mais prático,
    basicamente, ele nos provê uma maneira de criar chamadas HTTP, como se fôssemos um cliente acessando a API.
